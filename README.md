Проект находится в стадии активной разработки.


## **Ссылки на свёрстанные экраны**

```url http://localhost:3000/ ``` - домашний экран
```url http://localhost:3000/500 ``` - экран ошибки
```url http://localhost:3000/404 ``` - экран 404 ошибки(либо любой неизвестный путь)
```url http://localhost:3000/sign-in ``` - экран авторизации
```url http://localhost:3000/sign-up ``` - экран регистрации
    
**в дизайне экран настроек аккаунта реализован через модальное окно, поэтому его нет**
---

## Установка

- `npm install` — установка стабильной версии,
- `npm run dev` — запуск версии для разработчика,
- `npm run start` — сборка и запуск стабильной версии.

## Описание

Проект помогает людям коммуницировать друг с другом на расстоянии.
